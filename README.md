# Snowboard-Surfing
Carve through the slopes and defy gravity in this fast-paced 2D snowboard challenge.

Controls:

Right Arrow – Rotate clockwise

Left Arrow – Rotate counter-clockwise

Up Arrow – Boost forward

Down Arrow – Slow down

Objective:
Ride to the finish line without crashing.
But be careful — if your head hits the ground, it's game over.

Balance, boost, and land safely to win. Are you ready to master the mountain?
